# Etherbit Tiling — Central de Lançamentos (Releases)

Este repositório é utilizado exclusivamente para hospedar os arquivos de lançamento públicos (releases e binários) do **Etherbit Tiling**. 

O código-fonte original é mantido em um ecossistema privado para proteção de propriedade intelectual, mas você pode baixar e utilizar os executáveis compilados livremente por aqui.

## 🚀 Download e Novidades

Para baixar o aplicativo, ver o guia de instalação passo a passo e conferir todos os recursos e novidades da versão atual, acesse o nosso site oficial:

👉 **[etherbit-tiling.vercel.app](https://etherbit-tiling.vercel.app)**
